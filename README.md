# TravelEagle

#  Personalized, AI-assisted Travel Guidance and Navigation

## This is a student-led Senior Project at the New York Institute of Technology

---------------------------------------------------

## Set up Instructions:

1. Open terminal and clone the repository

2. Download the ExpoGo application on your mobile device

3. Download the latest version of Node.js on your computer

4. Open the terminal and type "**cd TravelEagle**"

5. Type "**npm install**" in the terminal to install all project dependencies for the application

6. Ensure your computer and mobile device are connected to the same internet connection

7a. Run "**npx expo start**" in the terminal to launch the application

  7b. If the application does not load on ExpoGo, try running: "**npx expo start --tunnel**"

8. Use your phone's camera to scan the QR code that will appear in the console. The QR code will open the application in the ExpoGo app
